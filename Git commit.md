Git commit:

Eftersom github är ett versionshanteringsverktyg så kan man säga att git commit är en version som anger vad som sparades just då i projektet. 
Man kan se vad som laddades upp, av vem, när, och genom en bra beskrivning av varje commit blir det enkelt att följa vad som hände när. 
Dessutom kan du gå tillbaka till en tidigare version om något fel har inträffat i den senaste versionen i main. 
Genom commit kan man dessutom på ett enkelt sätt samarbeta med andra utvecklare.
