Varför använder man github för att samarbeta? Vad är pull requests, branches och merge?

Man använder github för att samarbeta med andra då, 1 projekt man hjälper varandra inte ligger lokalt på någons dator utan ligger på githubsservrar. 
Genom att använda branches kan man koda på en helt egen del av projektet utan att det påverkar main.
Genom en pull request får du inte bra möjlighet att erbjuda din kod till arbetet (genom den bransch du skapade till dig själv) dessutom kan andra granska koden, ge sin feedback innan den eventuellt blir en del av main genom en merge.
